Getting started

Requierments:

Make sure that Java and Ant are installed: 
java -version
ant -version

Installing:

mkdir myapp
cd myapp
curl https://rawgit.com/ami-team/AMIWebFramework/master/tools/scripts/build.xml > build.xml
ant update_prod    # setup AWF in prod. mode
# or:
ant update_debug   # setup AWF in debug mode
Basic myapp/index.html: 
<?xml version="1.0" encoding="utf-8"?>

<!DOCTYPE html>

<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="en">
  <head>
    <meta charset="utf-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no" />

    <title>Template</title>

    <script type="text/javascript" src="js/jquery.min.js"></script>
    <script type="text/javascript" src="js/AMI/framework.min.js"></script>

    <script type="text/javascript">

	amiWebApp.onStart = function() {

		amiWebApp.autoRunSubApp('amiDocument');
	};

	amiWebApp.onToolbarUpdateNeeded = function() {

		$('#ami_menu_content').html('');
	};

    </script>

    <!--[if lt IE 9]>
      <script type="text/javascript" src="js/html5shiv.min.js"></script>
      <script type="text/javascript" src="js/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>

    <script type="text/javascript">

	amiCommand.endpoint = 'https://localhost:8443/AMI/FrontEnd';

	amiWebApp.start({
		logo_url: 'images/logo-alt.png',
		theme_url: 'twig/AMI/Theme/blue.twig',
	});

    </script>

  </body>
</html>
