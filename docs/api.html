Getting started

Requierments:

Make sure that Java and Ant are installed: 
java -version
ant -version

Installing:

  1. Download the installer: 
mkdir myapp
cd myapp
curl https://rawgit.com/ami-team/AMIWebFramework/master/tools/build.xml > build.xml
  2. Setup AWF in production mode or in debug mode: 
ant setup-prod    # setup AWF in prod. mode
# or:
ant setup-debug   # setup AWF in debug mode
  3. Create an index.html page. You will have to specify the title of the application and the AMI endpoint URL (e.g. https://localhost:8443/AMI/FrontEnd): 
ant new-index
